Input Files

 To run the analysis you will be asked to provide:

• Input 2 files, with the Phylogenetic Trees and Leaf States desired, if the ancestral states are not yet calculated. (e.g.)

File: Phylogenetic Trees

(A:1,(B:1,(E:1,D:1):0.5):0.5);




File: Leaf States

id,State
A,State1
B,State1
D,State3
E,State2



• Input directly a file with all the Ancestral States calculated. (e.g)

File: Ancestral States Calculated

Entity, State1, State2, State3
ROOT, 0.1,0.2,0.7
node_1, 0.333,0.333,0.334
node_2, 0,0.5,0.5
