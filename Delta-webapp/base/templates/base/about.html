About Delta-statistic

The website calculates the delta-statistic using one or multiple phylogenetic trees and ancestral states.

  • The ancestral states can be inputted directly, or they can be calculated using PastML, a software program commonly used for ancestral state reconstruction in phylogenetics.

  • Afterwards, the results can be generated and downloaded into a text or CSV file.

FAQ.

 This website was built using a combination of Django and Bootstrap. Django is a popular web framework for building scalable and maintainable websites, and was used to handle the server-side logic of the site. Bootstrap, a CSS framework, was used to create a responsive design that works well on devices of all sizes. JavaScript was also used to add some interactive elements to the site, such as drop-down menus. These technologies were chosen because of their versatility and ease of use functionalities, with the purpose of delivering a seamless user experience and offering a wide range of features.

 This website dedicated to calculating the delta-statistic was launched in the 30th of May 2022.

 The creators of this website have made the code available on GitHub that allows users to access the code and even make modifications or improvements to the site. By making the code open source, we hope to foster a community of developers and researchers who can collaborate and contribute to the project.
